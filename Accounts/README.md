# Finance Dashboard

A simple finance dashboard with sidebar navigation for managing incomes and expenses.

## Setup Instructions

### 1. Database Setup
- Import `database.sql` into your MySQL database
- Update database credentials in `api/config.php` if needed

### 2. Server Setup
- Place files in your web server directory (e.g., htdocs for XAMPP)
- Make sure PHP and MySQL are running

### 3. Access
- Open `index.html` in your browser through localhost

## Features
- Dashboard with financial statistics
- Income management (add/delete)
- Expense management (add/delete)
- Reports section (ready for customization)
- Responsive sidebar navigation
- AJAX-powered data updates

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP, AJAX
- Database: MySQL
